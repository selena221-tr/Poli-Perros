# REGISTRO DE POLI-PERROS 


Este programa permite ingresar la informacion de los PoliPerros: NOMBRE, HUELLA DACTILAR Y FOTO.


## Utilización

Puede acceder con mayor facilidad descargando el archivo desde el siguiente link: 
https://github.com/selena221-tr/Poli-Perros/blob/main/Programa_Polierros.exe

## Funcionalidades

- Registrar la información de un Poliperro
- Buscar Poliperros por su huella dactilar
- Actualizar información de un Poliperro
- Eliminar un registro
